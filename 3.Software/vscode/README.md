# 平衡小车操作说明

固件版本：**ver1.7.1**  
控制方式：Xbox 无线手柄（蓝牙）或上位机串口（`host_data`）

---

## 一、上电与站立

小车**上电后不会自动站立**，这是安全设计。需手动触发初始化。

### 操作步骤

1. 给小车上电，等待约 **1 秒** 完成系统初始化。
2. 通过蓝牙连接 Xbox 手柄（见下方「手柄配对」）。
3. 将小车扶稳，确保轮子着地、姿态正常。
4. 按 **RB（右肩键）** 触发站立流程。
5. 等待约 **1～3 秒**，小车自动完成伸腿、平衡恢复，进入正常平衡模式。

### 站立流程（自动）

| 阶段 | 说明 | 耗时 |
|------|------|------|
| 等待信号 | 舵机力矩关闭，等待 RB | 手动 |
| 伸腿 | 舵机移动到站立位置 | 即时 |
| 准备 | 关闭电机与平衡，等待稳定 | ~350 ms |
| 恢复 | 开启平衡控制，等待姿态稳定 | ~0.25～2.5 s |
| 完成 | 进入平衡模式，可正常遥控 | — |

## 二、手柄配对

默认蓝牙 MAC 地址配置在 `lib/devices/xbox_dev.cpp`：

```
65:D9:64:B2:08:60
```

若手柄无法连接，请将上述地址改为你手柄的实际 MAC 地址后重新烧录。

- 连接成功：手柄震动约 **1 秒**。
- 断开连接：手柄短震约 **0.2 秒**。
- 未连接手柄时，按键输入来自上位机串口（`host_data`）。

## 三、摇杆控制（平衡模式）

| 输入 | 功能 | 说明 |
|------|------|------|
| **右摇杆 ↑↓** | 前进 / 后退 | 最大线速度 0.6 m/s；后退速度为前进的 **80%** |
| **左摇杆 ←→** | 原地转向 | 最大角速度 4.0 rad/s |
| 摇杆死区 | — | 约 5%，小幅度偏移不会触发运动 |

## 四、按键功能总览

### 4.1 模式切换

| 按键 | 适用模式 | 功能 |
|------|----------|------|
| **RB** | 首次启动（等待信号） | 触发站立初始化 |
| **RB** | 坐下完成态 | 重新站起，返回平衡模式 |
| **LB** | 平衡模式 | 进入「坐下」模式 |
| **RS** | 平衡模式 | 原地跳跃 |
| **Y** | 平衡模式 | 向前跳跃 |
| **A** | 平衡模式 | 向后跳跃 |
| **X** | 平衡模式 | 向右转 90° 跳跃 |
| **B** | 平衡模式 | 向左转 90° 跳跃 |

> 跳跃动作执行完毕后会自动回到平衡模式。执行跳跃前请确保周围有足够空间。

### 4.2 姿态与腿部调节（平衡模式）

方向键需**单独按下**（不能与其他键同时按），每次调节步进 0.025。

| 按键 | 功能 |
|------|------|
| **↑** | 抬高机身（减小腿高基准） |
| **↓** | 降低机身（增大腿高基准） |
| **←** | 减小 Roll 补偿（机身左倾补偿） |
| **→** | 增大 Roll 补偿（机身右倾补偿） |

### 4.3 系统复位与力矩

| 按键 | 适用模式 | 功能 |
|------|----------|------|
| **LS** | 平衡模式 | 在**无前进/后退指令**时，重置 Roll 补偿、腿高基准及运动积分 |
| **LS** | 坐下完成态 | 关闭舵机力矩（腿变软，便于手动搬动） |

### 4.4 摄像头（平衡模式）

| 组合键 | 功能 |
|--------|------|
| **Select + ↑** | 摄像头向上转 |
| **Select + ↓** | 摄像头向下转 |

摄像头角度范围：0°～180°（具体朝向取决于安装方式，见 `ptk7350.h`）。

### 4.5 当前未使用的按键

以下按键在固件中**暂无功能**：

- Share、Start、Xbox 键
- LT / RT 扳机

## 五、工作模式说明

### 平衡模式（BALANCE）

默认工作模式。LQI 平衡控制开启，可通过摇杆行驶，可用方向键微调姿态。

### 坐下模式（SIT）

1. 平衡模式下按 **LB** 进入。
2. 小车后轮反转，机身前倾坐下。
3. 坐下完成后（或超时 5 秒）电机停止。
4. 可选按 **LS** 关闭舵机力矩。
5. 按 **RB** 重新站起。

### 跳跃模式（JUMP）

| 触发键 | 跳跃类型 |
|--------|----------|
| RS | 原地跳 |
| Y | 向前跳 |
| A | 向后跳 |
| X | 向右转 90° 跳 |
| B | 向左转 90° 跳 |

跳跃过程：收腿 → 蹬地 → 腾空 → 落地 → 姿态恢复 → 自动回平衡模式。

## 六、常见问题

| 现象 | 可能原因 | 处理建议 |
|------|----------|----------|
| 上电后不站立 | 正常，需按 RB | 连接手柄后按 RB |
| 按 RB 无反应 | 手柄未连接或 MAC 不对 | 检查蓝牙连接，修改 MAC 地址 |
| 站立后立刻倒 | 初始姿态不佳 | 上电前扶稳，确保轮子着地 |
| 方向键调节无效 | 同时按了其他键 | 单独按方向键 |
| LS 复位无效 | 右摇杆仍有前进/后退输入 | 松开右摇杆后再按 LS |
